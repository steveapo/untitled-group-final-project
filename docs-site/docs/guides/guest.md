# Guest Mode Guide

**Guest mode** is the entry point for new visitors. You can browse available rooms without creating an account.

## Starting as a Guest

From the main menu, select **"1. Continue as Guest"** to view room inventory without authentication.

## Typical Guest Journey

### Main Menu

```
╔══════════════════════════════════════╗

  ● R101   | Single   | $59.99/night  | AVAILABLE
  ● R102   | Double   | $89.99/night  | AVAILABLE
  ● R103   | Triple   | $119.99/night | AVAILABLE
  ● R104   | Quad     | $149.99/night | AVAILABLE
  ● R105   | Suite    | $199.99/night | MAINTENANCE
  ● R201   | Single   | $59.99/night  | AVAILABLE
  ● R202   | Double   | $89.99/night  | AVAILABLE

─────────────────────────────────────────
```

**Status Indicators:**
- **●** (green) = Room is available for booking
- **●** (red) = Room is under maintenance or occupied

## Room Details

### Room Types

The system offers five room types:

| Type | Typical Capacity | Features | Price Range |
|------|------------------|----------|------------|
| Single | 1 | Private room for one guest | $50-70 |
| Double | 2 | Standard room for couples/two guests | $80-120 |
| Triple | 3 | Room for three guests | $110-160 |
| Quad | 4 | Spacious room for small groups | $140-200 |
| Suite | 2+ | Premium room with amenities | $180-300 |

### Pricing

Nightly rates vary by room type. Total booking cost = **Nightly Rate × Number of Nights**

Example:
- Double room: $89.99/night
- 3-night stay: $89.99 × 3 = **$269.97 total**

## Next Steps

### To Browse More
Press any key to return to the Guest menu and select again, or press **Esc** to go back.

### To Book a Room
You must create an account first:

1. Select **"3. Register"** from Guest menu
2. Create your account with:
   - Username
   - First and last name
   - Email address
   - Strong password (8+ chars, uppercase, lowercase, number, symbol)
3. Log in with your credentials
4. Access the full User menu for booking

### To Check Availability by Date
After registering and logging in, use the **"2. Search and book a room"** option in the User menu to:
- See room availability for specific dates
- Use the interactive occupancy calendar
- Book directly from the calendar interface
